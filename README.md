# 🐾 Tienda de Mascotas (PetShop) en PHP y MySQL

Este es un proyecto educativo de una página web básica para una tienda de mascotas. Incluye un sistema de autenticación (Login), un carrito de compras funcional y persistencia de datos para guardar los pedidos en una base de datos.

## 🚀 Características
- **Login:** Acceso protegido con validación de usuarios.
- **Catálogo Dinámico:** Carga de productos directamente desde MySQL.
- **Carrito de Compras:** Suma subtotales y permite gestionar artículos en la sesión.
- **Historial de Pedidos:** Guarda las compras confirmadas de forma permanente en la BD.

## 🛠️ Requisitos e Instalación

1. Descarga e instala un servidor local como **XAMPP** o **WampServer**.
2. Clona este repositorio o descarga los archivos dentro de la carpeta `htdocs` (en XAMPP).
3. Crea una carpeta llamada `imagenes/` en la raíz del proyecto y añade las fotos correspondientes de los productos (`perro.jpg`, `gato.jpg`, etc.).
4. Abre **phpMyAdmin** (`http://localhost/phpmyadmin/`).
5. Crea una nueva base de datos llamada `tienda_mascotas`.
6. Ve a la pestaña **Importar** y sube el archivo `tienda_mascotas.sql` incluido en este proyecto.
